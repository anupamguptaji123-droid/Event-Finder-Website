
# 🎓 College Event Hub

A modern and user-friendly web application to manage and explore college events efficiently.

✨ Never miss an event again!


## 🌐 Live Demo

🔗 **View Project:**
👉 https://finder-sage-one.vercel.app/


## 🚀 Features

* 📅 **Featured Event Section**
* 🧾 **Upcoming Events Listing**
* ⏳ **Live Countdown Timer**
* 📝 **Event Registration Form**
* ➕ **Add New Event Option**
* 🎨 **Modern Gradient UI Design**

## 📌 How It Works

1. View events on homepage
2. Click **Register Now**
3. Fill the registration form
4. Track time using countdown timer

## 📊 Project Use Case (For College)

This project solves a **real-world problem**:

> Students often miss important college events due to lack of centralized information.

💡 **Solution:** A single platform to:

* View all events
* Register easily
* Track event timing

 📌 Usage
Click "Add New Event" to create an event
Browse events in Featured and Upcoming sections
Click "Register Now" to fill out the registration form
View time remaining for upcoming events
